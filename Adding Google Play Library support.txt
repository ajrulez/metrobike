Adding Google Play API

(1) Load the library into Eclipse
In Eclipse, File -> Import
  Android -> Existing Android Code Into Workspace
  from HelloMetroBike/libs/google-play-services_lib

(2) Add the library to the target project
In Eclipse, right click HelloMetroBike project -> Properties
  Under Android, Library, Add... google-play-services_lib

(3) Rebuild
In Eclipse, Project -> Clean... and clean all
